# Linux Kernel Semantic Patch Generator (LK-SPG) Agent

## Overview
This project implements an intelligent Agent system based on LLM to automate the generation of Coccinelle (SmPL) semantic patches for Linux Kernel upgrades.

## Architecture
- **Agent Framework**: LangGraph
- **Tooling**: Model Context Protocol (MCP)
- **Knowledge Base**: RAG with ChromaDB

## Prerequisites
- Python 3.10+
- `coccinelle` (`spatch`) installed on the system.

## Installation
```bash
pip install .
```
